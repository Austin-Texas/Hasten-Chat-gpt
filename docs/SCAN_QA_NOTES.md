# HASTEN Scan QA Notes

## Route

- `/driver/scan`

## Checks

- Scan page only lists loads assigned to the current driver.
- Load selector shows active assigned loads.
- BOL and POD toggle buttons work.
- Upload button accepts image or PDF files.
- Large file check appears before upload.
- OCR result saves a LoadDocument row.
- OCR mismatch warnings appear when extracted values differ from load values.

## Result

Driver document scanning should be tested on a mobile device before native app release.

# Store Release Checklist

## App identity

- App name: HASTEN Cargo
- Bundle ID / package name selected
- App icon ready
- Splash screen ready
- Light/dark theme checked

## Required policies

- Privacy policy URL
- Terms of service URL
- Support email
- Data safety answers
- Location usage explanation
- Camera/photo usage explanation
- Notification usage explanation

## Driver app checks

- Login works
- Driver dashboard opens
- Loads list opens
- Load detail opens
- Scan/upload works
- Offline upload retry is visible
- Profile opens
- Settlement preview opens

## Build checks

- Web build passes
- Android debug build opens
- Android release build signs correctly
- iPhone simulator opens
- iPhone archive validates

## Final note

Do not submit to stores until the production backend, privacy policy, and support contact are ready.

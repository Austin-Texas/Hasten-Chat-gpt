# Mobile Native Packaging Checkpoint

## Goal

Prepare HASTEN driver mobile workflow for Android and iPhone packaging.

## Required before packaging

- Driver dashboard opens.
- Driver loads list opens.
- Driver load detail opens.
- Driver scan opens.
- Driver messages opens.
- Driver profile opens.
- Driver settlement preview opens.
- Upload retry flow is visible.
- Build completes successfully.

## Current recommendation

Finish web/mobile hardening first, then package the same driver surface for native app release.

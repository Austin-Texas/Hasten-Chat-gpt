# Native Setup Steps

## Start only after web hardening passes

1. Apply recommended patches.
2. Run helper verification.
3. Run production build.
4. Review native packaging status.
5. Add Capacitor dependencies.
6. Add native Android project.
7. Add native iPhone project.
8. Test Android debug build.
9. Test iPhone simulator build.
10. Prepare store release assets.

## Required driver workflows

- Login
- Dashboard
- Loads
- Load detail
- Scan/upload
- Upload retry
- Messages
- Profile
- Settlement preview

## Important

Do not publish until backend, privacy policy, support email, and app store assets are ready.
